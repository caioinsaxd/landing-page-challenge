# landing-page-challenge

### Ferramentas de IA utilizadas
OpenCode (assistente de IA) e Claude Sonnet (modelo) para geração de código e debugging.

### Etapas onde a IA mais ajudou
Criação dos componentes, soluções de estilização com Tailwind, e implementação de animações.

### Ajustes manuais realizados
Ajuste fino de timing de animações para melhor UX, refinamento de cores para contraste e acessibilidade e etc. Houveram ajustes manuais em praticamente todos os componentes e animações. Utilizei a criação da IA apenas como rascunho e poli conforme a minha visão de forma manual.

### Stack utilizada
React + TypeScript + Vite + Tailwind CSS
