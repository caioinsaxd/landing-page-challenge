### Ferramentas de IA utilizadas e Stack
OpenCode (assistente de IA) e Claude Sonnet (modelo) para geração de código e debugging.
React + TypeScript + Vite + Tailwind CSS

### Etapas onde a IA mais ajudou e Ajustes manuais realizados
Criação dos componentes, soluções de estilização com Tailwind, e implementação de animações. Ajuste de timing de animações para melhor UX, refinamento de cores para contraste e acessibilidade e etc. Fiz ajustes manuais em praticamente todos os componentes e animações. Utilizei a criação da IA apenas como rascunho e poli conforme a minha visão de forma manual.
